# 🌍 Green House Gas Category Prediction System

An AI-powered web application built using Streamlit and Machine Learning that predicts greenhouse gas emission categories based on country, year, and emission values.

## 🚀 Features

- Interactive Streamlit Dashboard
- Real-time Greenhouse Gas Category Prediction
- User-Friendly Interface
- Country-wise Emission Analysis
- Prediction Confidence Score
- Fast and Lightweight Deployment

## 🛠️ Technologies Used

- Python
- Streamlit
- Scikit-learn
- Joblib
- NumPy
- Pandas

## 📊 Input Features

- Country
- Year
- Emission Value

## 🎯 Output

Predicts the greenhouse gas emission category such as:

- CO₂ Emissions
- Methane (CH₄) Emissions
- Nitrous Oxide (N₂O) Emissions
- Hydrofluorocarbons (HFCs)
- Perfluorocarbons (PFCs)
- Sulphur Hexafluoride (SF₆)
- And other greenhouse gas categories
